# ============================================
# ProductVault - Installation Steps
# ============================================

## Step 1: Extract ZIP
Extract `ProductVault` folder to:
```
C:\xampp\htdocs\takwincart\packages\workdo\ProductVault\
```

So the path looks like:
```
C:\xampp\htdocs\takwincart\packages\workdo\ProductVault\composer.json
C:\xampp\htdocs\takwincart\packages\workdo\ProductVault\module.json
C:\xampp\htdocs\takwincart\packages\workdo\ProductVault\src\...
```

## Step 2: Run Setup (Database + Permissions)
Open in browser:
```
http://localhost/takwincart/packages/workdo/ProductVault/setup.php
```

## Step 3: Clear Cache
Open CMD:
```
rmdir /s /q C:\xampp\htdocs\takwincart\storage\framework\views
rmdir /s /q C:\xampp\htdocs\takwincart\storage\framework\cache
```

## Step 4: Test
```
http://localhost/takwincart/product-vault
```
